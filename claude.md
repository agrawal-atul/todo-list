\# Todo List Project



\## Purpose

A simple task tracker to test git sync across machines.



\## Rules

\- Tasks are stored in `tasks.md`

\- Each task has: status \[ ] or \[x], description, date added

\- Never delete completed tasks — mark them done instead

\- When I say "add task: <text>", append it to tasks.md

\- When I say "done: <task>", mark it complete in tasks.md

\- When I say "show tasks", list all pending tasks
