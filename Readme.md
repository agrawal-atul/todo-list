\# Todo List — Cowork Test Project



This project tests git sync across machines.



\## How to use

Open this project in Cowork and say:

\- "add task: <description>" → adds a new task

\- "done: <task name>" → marks it complete

\- "show tasks" → lists pending tasks



\## Sync

git pull before starting, git push after finishing.
